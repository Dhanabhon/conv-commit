# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-06-16

### Added
- `conv-commit-gitmoji` skill for generating conventional commits prefixed with unicode gitmojis.

## [0.1.0] - 2026-06-16

### Added
- Initial release of the `conv-commit` plugin for AI agents.
- Core folder structure compatible with Codex Plugins.
- `conv-commit` skill for generating standard Conventional Commits.
- `conv-commit-concise` skill for generating ultra-brief, single-line commits.
- `conv-commit-detailed` skill for generating comprehensive commit messages with context, reasoning, and side-effects.
- `conv-commit-sarcastic` skill for generating sarcastic and passive-aggressive commit messages.
- Introduced `antigravity-rtk-rules.md` in `.agents/rules` for token optimization.
- Comprehensive `README.md` with installation instructions tailored for Codex CLI and Codex App via the GitHub repository.
